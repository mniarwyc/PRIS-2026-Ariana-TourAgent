# 🌍 TourAgent Pro: Интеллектуальная экспертная система подбора туров

## 📌 Описание проекта
TourAgent — это интеллектуальное приложение для автоматизации подбора туристических направлений. Система анализирует запросы пользователя на естественном языке, сопоставляет их с базой знаний и выдает персонализированные рекомендации.

В отличие от простых поисковых скриптов, проект реализует **гибридный подход**:
1.  **Граф знаний (Knowledge Graph):** Семантические связи между странами, городами и визовыми требованиями.
2.  **Экспертная логика (Expert System):** Фильтрация по бизнес-правилам, хранящимся в формате JSON.
3.  **Дата-центричность (Data-Driven):** Полное отделение данных от кода через использование внешних CSV-датасетов.

---

## 🛠 Технологический стек
Проект реализован на языке **Python** с использованием следующих библиотек:
- **Streamlit**: Веб-интерфейс и интерактивные дашборды.
- **Pandas**: Загрузка, фильтрация и аналитическая обработка датасета.
- **NetworkX**: Построение и визуализация графовых структур данных.
- **Matplotlib**: Генерация гистограмм цен и статистических отчетов.
- **Regex (re)**: Парсинг параметров (бюджета) из текстовых сообщений пользователя.

## 📐 Архитектура системы
```mermaid
graph TD;
    User[Пользователь] --> UI[Streamlit Interface];
    UI --> Logic[Expert Logic / Regex Parser];
    
    subgraph "Knowledge Core"
        Logic --> Graph[Knowledge Graph - NetworkX];
        Logic --> Rules[Rules Engine - JSON];
    end
    
    subgraph "Data Layer"
        Graph --> CSV[Dataset - Pandas];
    end
    
    UI --> Analytics[Analytics Dashboard];
    Analytics --> CSV;
    
    Logic --> UI;
🚀 Основной функционал
Интеллектуальный чат: Обработка запросов типа "хочу пляж до 800$" или "варианты в Италии".

Аналитический дашборд: Визуализация распределения цен и рейтингов по всей базе туров.

Визуализация графа: Интерактивная схема связей между объектами туризма.

Обработка исключений: Система предлагает альтернативные варианты, если точного совпадения по бюджету не найдено.

Экспорт данных: Возможность скачать текущий аналитический отчет в формате CSV.

📦 Инструкция по запуску
Клонируйте репозиторий:
git clone [https://github.com/ваш-логин/TourAgent.git](https://github.com/ваш-логин/TourAgent.git)
Установите необходимые библиотеки:
pip install streamlit pandas networkx matplotlib
Запустите приложение:
streamlit run src/main.py
