## 📌 Описание проекта
**TourAgent** — это интеллектуальное веб-приложение для автоматизации подбора туристических направлений. Система анализирует запросы пользователя на естественном языке, сопоставляет их с графом знаний и выдает персонализированные рекомендации.

В отличие от классических поисковых систем, проект реализует **гибридный подход**:
*   **Граф знаний (Knowledge Graph):** Семантические связи между странами, направлениями и уникальными характеристиками туров.
*   **Экспертная логика (Expert System):** Гибкая фильтрация по бизнес-правилам (бюджет, тип отдыха, сезонность).
*   **Data-Driven архитектура:** Полное отделение логики от данных (используются внешние CSV-наборы данных).

---

## 🛠 Технологический стек
*   **Streamlit:** Современный веб-интерфейс и интерактивные панели управления.
*   **Pandas:** Загрузка, фильтрация и многоуровневая обработка данных.
*   **NetworkX:** Построение, визуализация и навигация по графовым структурам.
*   **Regex (re):** Интеллектуальный парсинг интентов (извлечение бюджета) из сообщений пользователя.

## 📐 Архитектура системы
```mermaid
graph TD;
    User[Пользователь] --> UI[Streamlit Interface];
    UI --> Logic[Expert Logic / Regex Parser];

    end

🚀 Основной функционал
Интеллектуальный чат: Распознавание естественного языка (например, "хочу море до 1200$" или "варианты в горах").

Парсинг параметров: Автоматическое извлечение финансовых лимитов и предпочтений с помощью регулярных выражений.

Отказоустойчивость: Модуль загрузки данных защищен от ошибок кодировки (Unicode) и структурных аномалий в CSV-файлах.

Аналитический дашборд: Визуализация распределения цен и рейтингов для наглядного анализа базы туров.

Консультативный UI: Вывод структурированных карточек туров с изображениями и прямой связью с менеджером.

📦 Инструкция по запуску
Клонируйте репозиторий:

git clone [https://github.com/ваш-логин/TourAgent.git](https://github.com/ваш-логин/TourAgent.git)
cd TourAgent
Установите зависимости:

pip install streamlit pandas networkx matplotlib
Запустите приложение:

streamlit run src/main.py
